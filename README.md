# SeaSAGE-UNIV

SeaSAGE-UNIV: Semantic-Aware Guidance and Alignment for RGB-Infrared Maritime Detection.

This repository implements a semantic-aware RGB-Infrared maritime detection framework inspired by UNIV. The goal is to learn unified cross-modal representations for robust maritime object detection.

## Core Idea

- RGB-IR dual encoders based on UNIV / ViT
- Semantic label assignment for patch-level pseudo semantic guidance
- Semantic-aware PCCL for cross-modal patch alignment
- Unified RGB-IR representation for downstream maritime detection
- Detection heads including Faster R-CNN and YOLOv8-style head
